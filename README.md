# AI Startup Advisor

AI Startup Advisor is an intelligent platform that helps founders with startup planning, pricing strategies, marketing analysis, competitor analysis, and roadmap generation using AI-powered personalized responses.

## Features

* User Authentication (Login / Signup)
* AI Chatbot Interface
* Long-Term Memory System using Hindsight
* Dashboard & Analytics
* Pricing Suggestions
* Marketing Analysis
* Competitor Analysis
* Roadmap Generator
* Smart AI Model Routing
* Responsive UI/UX Design


## Tech Stack

### Frontend

* React.js

### Backend

* Node.js
* Express.js

### Database

* MongoDB / Firebase

### AI & Runtime

* cascadeflow
* Groq
* Ollama
* Hindsight Memory System
